# Singnam Aluminum Forecast App

## Giới thiệu
Dự báo giá nhôm LME trong 7 ngày, 1 tháng, 3 tháng. Phân tích AI tự động.

## Cài đặt
1. Cài Python 3.9+
2. pip install -r requirements.txt
3. export OPENAI_API_KEY="your_openai_api_key"
4. python app.py

## Deploy Railway
Fork và deploy tự động.
